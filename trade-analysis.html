📐 行为模式数学总结

📌 盈利三要素

  • ✔ 主线情绪爆发期（盈利率 +15% ~ +98%）
  • ✔ 龙头股/辨识度标的（流动性溢价）
  • ✔ 重仓 + 快速止损（尽管做得不彻底）

📌 亏损三特征

  • ✘ 非主流、冷门股（胜率仅30%左右）
  • ✘ 高位接力末期（天地板/连续A杀）
  • ✘ 亏损后加仓/不止损（亏损扩大3~5倍）
亏损分布
单笔亏损＞10% 的笔数占亏损总额的≈65%
盈利＞10% 的笔数仅占盈利总额的≈80%

结论：几笔大赚映射高光，几笔大亏摧毁账户 → 盈亏比严重不对称。

🔍 数据验证：你的账户波动根源 不在选股胜率，而在极端亏损控制。若剔除前10大亏损，整体收益率将大幅转正。

🛠️ 如何改进 · 稳定盈利实操手册

1️⃣ 彻底砍掉亏损源头

  • ❌ 放弃所有非主线、冷门小票（市值<50亿且非热点不参与）
  • ❌ 放弃转债高频交易（胜率不足35%，摩擦成本高）
  • ❌ 禁止任何形式的“亏损加仓”

2️⃣ 建立强制止损规则（必须执行）

  • 🚨 单笔亏损 -5% 无条件减仓50%
  • 🚨 单笔亏损 -8% 无条件清仓（哪怕盘中拉起）
  • 🚨 单日总亏损 ≥3%，停止交易，复盘24小时

3️⃣ 仓位管理纪律

  • 📦 单票上限 ≤ 总资金 15%（主升浪可放宽至20%，但需严格止损）
  • 📦 同一板块 ≤ 30% 仓位，避免黑天鹅
  • 📦 情绪退潮期整体仓位 ≤ 30%

4️⃣ 只做两类模式（重复你的优势）

🔥 模式A：主线龙头主升浪
- 识别政策/事件驱动的板块（军工/AI/华为链/金融）
- 买入板块最强辨识度标的（成交额排名前3）
- 持有至板块第一次大分歧，分批止盈
⚡ 模式B：情绪冰点转势接力
- 市场连板高度压缩至2板后寻找破局者
- 轻仓试错 → 确认转强加仓 → 断板即走

5️⃣ 每日复盘清单（强制）

  • ✅ 今日交易是否属于上述两种模式？
  • ✅ 止损位是否提前设置？（价格预警）
  • ✅ 有无情绪化/非计划交易？
  • ✅ 盈亏归因：盈利是否来自模式，亏损是否来自模式外
📌 核心心法：你不是选股能力不足，而是控制回撤的能力太弱。把“不亏大钱”放在第一位，复利自然发生。
