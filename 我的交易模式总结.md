## 一、我的核心模式

### 🎯 主升龙头战法

**选股标准：**
- 有主线不做支线，有总龙不做分支龙
- 只做历史股性好的股票
- 前期足够强打出辨识度
- 具有带动性，分歧时强

**买入节点：**
1. 情绪转冷买（买错也要买）
2. 节点买入很重要，忍住手等节点
3. 板块爆发日第一天先上车
4. 龙头分歧靠近5日线加仓（1/3~1/2仓位）

**卖出纪律：**
1. 情绪转弱卖（卖错也要卖）
2. 竞价不及预期直接跑
3. 高位只减仓不加仓
4. 拉红就走，卖飞就卖飞

### 💡 我的竞价法则

```
明牌焦点股竞价规则：
├── 超预期 → 不要轻易卖
├── 不及预期 → 该卖就卖，不要意淫
└── 没红开 → 窜刚股直接竞价跑

判断优先级：
1. 地位未定时 → 竞价重要（谁高谁可能成龙头）
2. 地位清晰时 → 竞价没那么重要
```

根据抖音实盘博主股海贼王的操作特点：

## 三、我的优化模式（融合版）

### ✅ 龙空龙 + 龙头战法

```
主升期 → 满仓龙头 → 高位震荡减仓 → 主跌期空仓 → 低位试错打首板 → 循环
```

### ✅ 节点战法

| 情绪阶段 | 操作 | 仓位 |
|---------|------|------|
| 情绪冰点 | 试错首板 | 1/4 |
| 情绪转暖 | 上车龙头 | 1/2~满仓 |
| 情绪高潮 | 减仓兑现 | 1/4 |
| 情绪转弱 | 清仓观望 | 空仓 |

### ✅ 预期差思维

> 生于分歧，死于一致！
> 
> 当市场大多数人希望它怎样时，它往往不按大多数人的想法走

## 五、风控铁律

### 止损纪律
- 破5日线止损（妖股龙头）
- 低开+无地位连板股=下杀直接走
- 不狠下心来永远做不大

### 仓位管理
- 高位不加仓，只减仓
- 分歧靠近5日线加仓（1/3~1/2）
- 满仓只在主升期龙头确认时

### 心态管理
- 做短线要杀伐果断
- 犹豫是大忌
- 管不住手=剁手！
